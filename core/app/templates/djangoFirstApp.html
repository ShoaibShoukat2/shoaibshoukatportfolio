Complete Step-up of Your First Django Project With Simple Steps

Posted on January 1, 2024 by Shoaib

Are you diving into the world of web development with Django? Fantastic choice! Django is a high-level Python web framework that simplifies the process of building web applications. In this beginner-friendly guide, we'll walk you through the essential steps to create and set up your first Django project and application. Whether you're new to programming or transitioning from another language, these steps will serve as your roadmap to success.

Django Project Setup Tutorial

Below are the steps to set up a Django project:

  1. Opening PowerShell:

    2. The initial step involves accessing PowerShell. To initiate this process, press the Windows key on your keyboard, then type 'PowerShell' into the search bar. Subsequently, right-click on the PowerShell icon that appears and opt for 'Run as Administrator' from the contextual menu.




    If Python is not installed on your system, you can download and install it from python.org.

  2. Install Python:
    • Make sure you have Python installed on your system.
  3. Install Django:
    • Use pip to install Django - pip install Django.
  4. Create a Django project:
    • Run django-admin startproject myproject.
  5. Configure static files:
    • Create a directory named static and define static file settings in settings.py.
    
                            myproject/
                            ├── myproject/
                            │   ├── __init__.py
                            │   ├── settings.py
                            │   ├── urls.py
                            │   └── wsgi.py
                            └── static/
                            
    
                            STATIC_URL = '/static/'
                            STATICFILES_DIRS = [
                                BASE_DIR / "static",
                            ]
  6. Configure templates:
    • Create a directory named templates and define template settings in settings.py.
    
                            myproject/
                            ├── myproject/
                            │   ├── __init__.py
                            │   ├── settings.py
                            │   ├── urls.py
                            │   └── wsgi.py
                            ├── static/
                            └── templates/
                            
    
                                TEMPLATES = [
                                {
                                    ...
                                    'DIRS': [os.path.join(BASE_DIR, 'templates')],
                                    ...
                                },

                                ]
  7. Run the development server:.
    • Start the server using python manage.py runserver.
  8. Create a Django app (optional):
    • Run python manage.py startapp myapp.
  9. Develop your app:
    • Create models, views, and templates to build your web application.
